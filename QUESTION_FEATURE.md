# 提问功能使用说明

## 功能概述

在分析结果的每个模块右侧，现在都有一个"💬 提问"按钮，允许你针对该模块的内容向 DeepSeek AI 提问。

## 支持提问的模块

- 项目概述
- 项目需求
- 可行性分析
- 技术栈建议
- 开发周期和步骤
- 风险分析
- 项目成员分析
- 报价分析

## 使用方法

1. 在分析结果页面，找到你想深入了解的模块
2. 点击该模块右侧的"💬 提问"按钮
3. 在弹出的对话框中输入你的问题
4. 点击"提问"按钮
5. AI 会基于当前的分析结果给出专业的回答

## 示例问题

### 项目需求模块
- "这些功能需求中，哪些是核心功能，哪些可以作为二期开发？"
- "非功能需求中的性能要求具体应该达到什么标准？"

### 技术栈建议模块
- "为什么选择 React 而不是 Vue？"
- "这个技术栈的学习成本如何？"
- "有没有更轻量级的替代方案？"

### 开发周期和步骤模块
- "如果要缩短开发周期，可以优化哪些环节？"
- "各个阶段的人力投入应该如何分配？"

### 报价分析模块
- "如果预算有限，可以如何调整方案？"
- "维护成本主要包括哪些方面？"

### 项目成员分析模块
- "如果只有 3 个开发人员，应该如何分配角色？"
- "是否可以外包部分工作来降低成本？"

## 技术实现

- 前端：React + CSS
- 后端：Express.js
- AI：DeepSeek API
- 对话框支持多轮提问（在同一个对话框中可以继续提问）

## 注意事项

- 提问需要消耗 DeepSeek API 额度
- AI 的回答基于当前的分析结果，确保分析结果准确完整
- 建议提出具体、明确的问题以获得更好的回答
